# COST - Club of Science and Technology Website

**Club of Science and Technology (COST)** adalah ekskul yang fokus pada pengembangan skill di bidang teknologi dan sains dari SMAN 1 Cicurug.

## 🚀 Teknologi

- **Framework:** Next.js 14 (React + TypeScript)
- **Styling:** Tailwind CSS
- **Database:** MongoDB
- **Authentication:** NextAuth.js
- **File Storage:** Cloudinary
- **Hosting:** Vercel

## 📋 Fitur

- ✅ Halaman Beranda (Hero, Featured Events, About)
- ✅ Sistem Autentikasi (Login, Register)
- ✅ Member Dashboard
- ✅ Admin Dashboard
- 🔄 Manajemen Kegiatan (Events)
- 🔄 Blog & Artikel
- 🔄 Galeri Foto
- 🔄 Pengumuman
- 🔄 Manajemen Member

## 🛠️ Setup & Installation

### Prerequisites
- Node.js 18+ dan npm
- MongoDB Atlas Account
- Cloudinary Account

### Installation

1. Clone repository:
   ```bash
   git clone https://github.com/sir-liyra/cost_sman1cicurug.git
   cd cost_sman1cicurug
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup environment variables:
   ```bash
   cp .env.example .env.local
   ```

4. Run development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) di browser
